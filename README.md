# Smart Hospital Management System (DSE Exp 10)

A Java implementation demonstrating the use of hybrid data structures to optimize hospital record management.

## Data Structures Used
* **HashMap:** Used for `O(1)` patient lookup by ID.
* **LinkedList:** Used for maintaining a dynamic list of records for sequential traversal.
* **Binary Search Tree (BST):** Used to maintain and display patient IDs in sorted order.

## How to Run
1. Ensure you have JDK installed.
2. Compile: `javac HospitalManagementSystem.java`
3. Run: `java HospitalManagementSystem`
